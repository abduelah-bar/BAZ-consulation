## Image Sizing Guide

To ensure optimal website performance and visual quality, please use the following dimensions and aspect ratios when sourcing or creating images.

| Page Location | Image ID (`placeholder-images.json`) | Recommended Dimensions | Aspect Ratio | Notes |
| :--- | :--- | :--- | :--- | :--- |
| **Homepage Hero** | `hero` | 1920 x 1080 px | 16:9 | This is a full-screen background image. High resolution is key. |
| **Homepage About** | `about-program` | 800 x 600 px | 4:3 | Standard content image, appears next to text. |
| **Homepage CTA**| `cta-section` | 800 x 600 px | 4:3 | Standard content image, appears next to text. |
| **About Us Hero** | `about-us-hero` | 1920 x 400 px | ~21:9 (Wide) | This is a wide banner-style hero image. |
| **About Us Page** | `our-company` | 800 x 600 px | 4:3 | Standard content image, appears next to text. |
| **About Us Page** | `our-vision` | 800 x 600 px | 4:3 | Standard content image, appears next to text. |
| **Services Hero** | `services-hero` | 1920 x 400 px | ~21:9 (Wide) | This is a wide banner-style hero image. |
| **Services Page** | Service Images (`service-business-setup`, etc.) | 800 x 600 px | 4:3 | These are the main content images on the services list page. |
| **Service Detail Hero** | `service-page-hero` | 1920 x 400 px | ~21:9 (Wide) | A unified, wide banner for all individual service pages. |
| **Service Detail Pages** | Content Images (`service-business-setup`, etc.) | 800 x 600 px | 4:3 | Standard content image, appears next to text on detail pages. |
| **FAQ Page Hero** | `faq-hero` | 1920 x 400 px | ~21:9 (Wide) | This is a wide banner-style hero image. |
